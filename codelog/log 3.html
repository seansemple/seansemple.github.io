How does a CSS media query work, and why is it useful?

A CSS media query is something you add after @media. This enables the browsers to only apply styles inside if the viewport or the device you are using matches.
This is instrumental in responsive design.

What "media features" are supported by media queries — such as "orientation" or "max-width" — and what do they do? Pick any 4 to explain.

Max-width/min-width: Target screens with dimensional values and change layouts at breakpoints.
Pointer: Tells us if the primary pointer is precise or imprecise so you can adjust hit targets or effects that hover.
Orientation: Determines if viewport is in portrait or landscape so it can change the style accordingly.
Prefers-reduced-motion: Determines if the user prefers less animation, thereby reducing motion effects to improve user accessibility.
