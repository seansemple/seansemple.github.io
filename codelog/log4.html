What These Properties Do


Color: sets the text color.
Background-color: fills the element's background with a specific color.
Border Color: sets the color of the element's border lines.

Ways CSS Accepts Color + Examples

Named Color: use keywords like "aliceblue" or "rebeccapurple"
Hexadecimal: "#ff0000" or shorthand "#f00"
RGB: rgb(255, 0, 153) (or modern form rgb(255 0 153 / 80%)) Uses color spaces and numbers to determine color specificity.
Other formats may include HSL: hsl(150, 30%, 60%); HWB: hwb(210 20% 10%); Lab: lab(60% 20 30)
