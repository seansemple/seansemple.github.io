Q3) Dimensions vs resolution (one sentence)


Dimensions are the width and height in pixels and resolution is how many pixels there are per unit length.

Extra credit: formats (GIF, PNG, JPG/JPEG, WebP, SVG)


GIF: supports simple animations, limited colors, file size can be bigger than newer formats.
PNG: lossless, supports transparency, great for crisp edges and UI art.
JPG/JPEG: lossy, great for photos, smaller than PNG for photos but not ideal for sharp graphics or text.
WebP: modern format, supports lossy and lossless, transparency, and animation, often smaller than JPG/PNG for similar quality.
SVG: vector graphics, scale cleanly at any size, great for icons, logos, and diagrams.
